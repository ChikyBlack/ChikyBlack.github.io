Fotografía Video Audio

HTML permite la integración de varios tipos de contenido multimedia, como imágenes, videos, audios, etc. Para resaltar información que se proporciona en las páginas web, y enriquecer el contenido de estas.

Además, HTML5 también ofrece características avanzadas para multimedia, como la manipulación de gráficos vectoriales con SVG (Scalable Vector Graphics) y la API de canvas para gráficos dinámicos y animaciones.
